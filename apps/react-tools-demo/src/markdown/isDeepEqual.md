# isDeepEqual
It returns true if the params are equal in depth. [See demo](https://react-tools.ndria.dev/#/utils/isDeepEqual)

## API

```tsx
isDeepEqual(objA: unknown, objB: unknown, map = new WeakMap()):boolean
```


> ### Params
>
> - __objA__: _unknown_
> - __objB__: _unknown_
> - __map=new WeakMap()?__: _WeakMap_
>



> ### Returns
>
> __result__
> - _boolean_  
>
