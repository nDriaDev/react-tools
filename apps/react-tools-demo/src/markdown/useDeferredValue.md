# useDeferredValue
_useDeferredValue_ hook polyfilled for React versions below 18. [See demo](https://nDriaDev.io/react-tools/#/hooks/lifecycle/useDeferredValue)

## API

```tsx
useDeferredValue<T>(value: T): T
```

> ### Params
>
> - __value__: _T_
>

> ### Returns
>
> __deferred value__
> - _T_  
>
