# isMouseEvent
It returns true if the event param is of MouseEvent type. [See demo](https://react-tools.ndria.dev/#/utils/isMouseEvent)

## API

```tsx
isMouseEvent(event: SyntheticEvent): boolean
```


> ### Params
>
> - __event__: _SyntheticEvent_
>



> ### Returns
>
> __result__
> - _boolean_  
>
