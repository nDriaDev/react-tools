# isTouchEvent
It returns true if the event param is of TouchEvent type. [See demo](https://ndriadev.github.io/react-tools/#/utils/isTouchEvent)

## API

```tsx
isTouchEvent(event: SyntheticEvent | Event): boolean
```

> ### Params
>
> - __event__: _SyntheticEvent_
>


> ### Returns
>
> __result__
> - _boolean_  
>
