# isShallowEqual
It returns true if the params are equal until first level depth. [See demo](https://nDriaDev.io/react-tools/#/utils/isShallowEqual)

## API

```tsx
isShallowEqual(objA: unknown, objB: unknown): boolean
```

> ### Params
>
> - __objA__: _unknown_
> - __objB__: _unknown_
>

> ### Returns
>
> __result__
> - _boolean_  
>
